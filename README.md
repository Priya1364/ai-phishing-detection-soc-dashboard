🔐 AI-Based Phishing Detection & SOC Dashboard

📌 Overview

This project is an advanced cybersecurity application that detects phishing messages using Machine Learning and simulates a Security Operations Center (SOC) dashboard.

It provides real-time threat analysis, risk scoring, and actionable insights.

---

🚀 Features

- 🔍 Phishing detection using ML (Naive Bayes + TF-IDF)
- 📊 Real-time risk scoring dashboard
- 🧠 Explainable threat intelligence
- 📁 SOC alert history tracking
- ⬇️ Downloadable reports
- 🎨 Professional UI with dark theme
